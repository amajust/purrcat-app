# PurrCat Application

This repository contains the source code for the PurrCat application.

PurrCat is a platform designed to connect pet owners (specifically cat owners), professional breeders, and service providers (groomers, drivers). The application aims to provide a trusted community, premium supplies, emergency veterinary services, and efficient management tools for all stakeholders.

## Key Features (Based on initial requirements):

### User-facing (End Users like Siska):
*   Access to trusted community and content (articles, events).
*   Ability to find and book services (e.g., grooming, veterinary).
*   Social features (sharing photos, "Paws Up").
*   Real-time tracking of service providers.

### Merchant-facing (Breeders like Pak Iwan):
*   Verification badge system for trustworthiness.
*   Professional sales dashboard.
*   Efficient vaccine tracking.

### Partner-facing (Groomers/Drivers like Bimo):
*   Real-time job alerts and GPS navigation.
*   Quick payout mechanisms.

### Admin-facing (Internal Ops like Andre):
*   **Content Management System (CMS):** Rich text editor, event scheduler, SEO management.
*   **Partner Verification (KYC Engine):** Document viewer, approval/rejection workflows.
*   **Financial & Analytics Dashboard:** Transaction monitoring, audit trail.
